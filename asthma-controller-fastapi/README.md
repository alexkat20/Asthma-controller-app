# Asthma Controller FastAPI

This project is a FastAPI application designed to help users manage their asthma-related data, including peak flow readings, medication management, and user information. The application provides a RESTful API for logging readings, managing medicines, and handling user registrations.

## Project Structure

```
asthma-controller-fastapi
├── app
│   ├── main.py                # Entry point of the FastAPI application
│   ├── routers                 # Contains route definitions for different functionalities
│   │   ├── readings.py         # Routes for peak flow readings
│   │   ├── medicine.py         # Routes for managing medicines
│   │   ├── users.py            # Routes for user management
│   │   └── uploads.py          # Routes for handling file uploads
│   ├── models                  # Contains database models and schemas
│   │   ├── __init__.py         # Initializes the models package
│   │   ├── database.py         # Database connection and session management
│   │   ├── schemas.py          # Pydantic models for request and response validation
│   │   ├── models.py           # SQLAlchemy ORM models for database tables
│   │   └── crud.py             # Database operations (CRUD) related to the models
│   ├── services                # Contains business logic and services
│   │   ├── notifications.py     # Functions for sending notifications
│   │   ├── analysis.py          # Functions for analyzing peak flow data
│   │   └── plotting.py          # Functions for generating plots
│   └── utils.py                # Utility functions for the application
├── alembic                     # Directory for database migrations
│   ├── env.py                  # Alembic environment configuration
│   ├── script.py.mako          # Template for generating migration scripts
│   └── versions                # Contains migration scripts generated by Alembic
├── .env                        # Environment variables for configuration
├── requirements.txt            # Lists dependencies required for the FastAPI application
├── pyproject.toml             # Configuration file for the project
├── .gitignore                  # Specifies files and directories to be ignored by Git
└── README.md                   # Documentation for the project
```

## Setup Instructions

1. **Clone the repository:**
   ```
   git clone <repository-url>
   cd asthma-controller-fastapi
   ```

2. **Create a virtual environment:**
   ```
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

3. **Install dependencies:**
   ```
   pip install -r requirements.txt
   ```

4. **Set up environment variables:**
   Create a `.env` file in the root directory and add your configuration variables, such as database connection strings and API tokens.

5. **Run the application:**
   ```
   uvicorn app.main:app --reload
   ```

6. **Access the API:**
   Open your browser and navigate to `http://127.0.0.1:8000/docs` to view the interactive API documentation.

## Usage

- **Log Peak Flow Readings:** Use the `/readings` endpoint to log and retrieve peak flow readings.
- **Manage Medicines:** Use the `/medicine` endpoint to add, update, and retrieve medicine information.
- **User Management:** Use the `/users` endpoint for user registration and updates.
- **File Uploads:** Use the `/uploads` endpoint to handle file uploads for bulk data processing.
