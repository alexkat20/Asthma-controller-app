# Asthma-controller-app
This App is designed to control asthma:
1) Log daily peak flow metry
2) Analyse peak flow metry
3) Plot graphics
4) Predict today's values
5) Send notifications about logging peak flow metry
6) Upload patients' data from csv or excel
7) Deep analysis (In progress)
